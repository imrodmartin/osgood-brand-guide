Communication Style

Voice & Tone

Osgood Indiana communicates as a trusted small-town government resource. The tone is welcoming, factual, and community-focused without being overly formal or bureaucratic.

Civic & Trustworthy Clear & Direct Community-Centered Approachable Practical
Do — Voice Principles
  • Use plain language — write for residents, not lawyers
  • Lead with the action: "Pay your bill online" not "Bill payment is available"
  • Be specific with times, dates, and phone numbers
  • Use "Town of Osgood" or "the Town" consistently
  • Celebrate community events with warmth (Bricktoberfest, Osgood Trails)
  • Keep emergency and service info prominent and unambiguous
Don't — Avoid These
  • Bureaucratic jargon or overly formal tone
  • Vague calls to action ("Click here", "Learn more" without context)
  • Long paragraphs for information residents need quickly
  • Corporate or marketing language
  • Outdated event info left on the site
  • Inconsistent use of department names or contact numbers

Platform & Implementation

Tech Stack

The site runs on Drupal 10 with the Bootstrap 5 contrib theme and DXPR Theme sub-theme. Page layouts are built with Layout Builder — no page builder outside core is required.

CMS
Drupal 10
Core CMS platform. Content types include article, page, business, landing_page, news, veterans, council_minutes.
Theme
Bootstrap 5 (contrib)
Base theme providing Bootstrap 5 grid and component support. Sub-themed with DXPR Theme for layout and style overrides.
Sub-theme
DXPR Theme
Provides the header region, sticky nav, full-screen search, and Layout Builder-compatible regions. Block classes: block-dxpr-theme-*.
Layout
Layout Builder (core)
All pages built with Drupal core Layout Builder. Block types include: basic, card, large_banner, stat_card, heading, video, image_embed.
Styles Module
Bootstrap Styles
Adds utility CSS classes in Layout Builder UI. Key classes: bs-bg-pine, bs-text-white, bs-p-3, bs-border-width-2, rounded.
Custom Colors
Bootstrap Color Override
Bootstrap 5 CSS variables overridden at :root. Custom brand colors applied via the bs-example-classes from the Bootstrap Styles module.
Fonts
Google Fonts
Bebas Neue, Open Sans, Source Sans 3. Loaded via <link> in theme head. No self-hosting detected.
Modules
Admin Toolbar + More
admin_toolbar, back_to_top, better_social_sharing_buttons, bootstrap_styles detected in stylesheet loads.
